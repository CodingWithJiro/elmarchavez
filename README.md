## How to Run

Open a terminal and type:

```bash
git clone https://github.com/CodingWithJiro/elmarchavez.git
cd elmarchavez
npm install
npm run dev
```

---

## Testing and Coverage

Run tests:

```bash
npm test
```

Run tests with coverage:

```bash
npm test -- --coverage
```
